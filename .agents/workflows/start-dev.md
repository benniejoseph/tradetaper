---
description: Start the local development environment
---
This workflow provides instructions for starting up the backend, frontend, and admin local development servers.

1. Start the NestJS backend API watch server
```bash
cd tradetaper-backend && npm run start:dev
```

2. Start the Frontend Next.js app in development mode
```bash
cd tradetaper-frontend && npm run dev
```

3. Start the Admin Dashboard Next.js app in development mode
```bash
cd tradetaper-admin && npm run dev
```
